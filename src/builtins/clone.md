##### Clone Functions

```alan
// Explicitly duplicates the provided value and returns it, useful when you don't want to mutate
// the original value
clone(any): any
clone(Array<any>): Array<any>
clone(void): void
clone(int8): int8
clone(int16): int16
clone(int32): int32
clone(int64): int64
clone(float32): float32
clone(float64): float64
clone(bool): bool
clone(string): string
```
